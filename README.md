# 15.-css-background-filter


1.background
      background: tomato url('saruman.png') no-repeat fixed center;같은 축약형으로 사용 가능함.

2. filter
      transition: all 1s 는 태그를 점차적으로 1s 후변경해준다.
      filter 접두어: 
      -webkit = 구글, 사파리에 적용
      -o = 오페라에 적용
      -moz = 파폭에 적용
      -ms = 익스플로러에 적용(대부분 생략)
